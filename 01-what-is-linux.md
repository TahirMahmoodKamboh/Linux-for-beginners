## ğŸ§  What Is Linux?

Linux is an **operating system (OS)**.

An operating system is software that:
- Starts your computer
- Manages hardware (CPU, RAM, Disk)
- Allows you to run applications
- Acts as a bridge between **user** and **hardware**

Examples of operating systems:
- Windows
- macOS
- Linux
